# Cutie Project Vision

Cutie is a personality-first humanoid robot platform.

The first version will serve as the Phoenix Lab assistant. She will help explain projects, support lab interaction, provide system awareness, and make human-robot communication feel more natural through a witty and funny personality.

Cutie's humor is a core human-robot interaction strategy designed to make people feel more comfortable, curious, and engaged when interacting with robots.


## Short-Term Goal

Build Cutie as a lab-assistant humanoid head with:

- Voice interaction
- Expressive face behavior
- Camera perception
- Speaker output
- Microphone input
- System health awareness
- Phoenix Lab knowledge
- Witty and funny conversation behavior

## Long-Term Goal

Evolve Cutie into a full humanoid robot platform capable of:

- Mobility
- Perception
- Manipulation
- Expressive communication
- Human-aware interaction
- Public interaction
- Whole-body humanoid coordination
