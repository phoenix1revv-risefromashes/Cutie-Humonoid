# Cutie System Architecture

Cutie is designed as a modular ROS2 humanoid robot platform.

The first version focuses on the lab-assistant head and interaction core. The architecture is designed to grow into future mobility, manipulation, and full humanoid embodiment.

## Core Layers

- Expressive face behavior
